# Rock Paper Scissor React App

## Description
This is a react based rock paper scissor game where the user plays against the computer. This application uses react components, state and hooks to create an interactive experience.

## Feautures
User selects one of the choices by clicking the image
The selected image is highlighted 
Computer randomly selects a throw after an animation
Computer initially shows a question mark
Displays the result win, loose, tie
Scoreboard tracks of win, loose, and tie
Reset button clears the game state and score.

# Technologies used
React (Vite)
Javascript
CSS

# Reflection
In this project I learned how to use React components state and hooks to build an interactive application. I also practice organizing code into reusable components and handling user events.

The most difficult part was implementing the computer animation and timing logic but using setInterval and setTimeout helped achieve the desire behavior.

Overall this project was helpful to reinforce my understandin of React fundamentals and building dynamic user interface.
